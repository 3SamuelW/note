## 📈 关于这个网站

### 技术栈

这个网站使用以下技术构建：

- 📝 **[MkDocs](https://www.mkdocs.org/)** - 静态网站生成器
- 🎨 **[Material for MkDocs](https://squidfunk.github.io/mkdocs-material/)** - 现代化主题
- 🏠 **[GitHub Pages](https://pages.github.com/)** - 免费托管平台
- ⚙️ **GitHub Actions** - 自动化部署

### 网站特色

- 🌓 支持深色/浅色模式切换
- 📱 完全响应式设计
- 🔍 全站搜索功能
- 🏷️ 智能标签系统
- ⚡ 快速加载和导航
